26/**
27 * Provides role assignment and checking capabilities to Eloquent models.
28 *
29 * This trait enables models (typically User) to be assigned roles, check role
30 * membership, and query by roles. Roles are managed through polymorphic relationships
31 * and automatically cleaned up when models are hard-deleted.

43     * Boot the HasRoles trait for model lifecycle events.
44     *
45     * Registers a model deletion listener that automatically detaches all assigned
46     * roles when the model is hard-deleted (not soft-deleted). This ensures referential
47     * integrity in the assigned_roles pivot table.
48     */
49    public static function bootHasRoles(): void
50    {
